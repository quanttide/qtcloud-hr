# 量潮人力资源云

QtCloud HR —— 人力资源领域产品，覆盖招聘、薪资核算等业务。

## 产品模块

| 模块 | 说明 |
|------|------|
| **招聘** | AI 辅助简历初筛、筛选标准配置、筛选结果反馈优化 |
| **薪资核算** | 计时工资试算、规则参数配置、差异化策略、自定义薪资项 |
